# Kanoe template

Simple basic project using:

- [Parcel](https://github.com/parcel-bundler/parcel)
- [PUG](https://github.com/pugjs/pug)
- [Stylus](https://github.com/stylus/stylus)

## Installation

```bash
npm install
```

## Usage

```bash
# development server
npm run start
# build to production
npm run build
```
